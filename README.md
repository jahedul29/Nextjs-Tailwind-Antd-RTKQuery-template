# Nextjs-Tailwind-Antd-RTKQuery-template

## Technologies used

- Nextjs v 14 (App router)
- Tailwind css
- Ant design
- Redux toolkit
- React hook form
- yup

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
